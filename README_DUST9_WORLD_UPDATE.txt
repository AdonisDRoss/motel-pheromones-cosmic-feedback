# Dust-9 World Update Package

Drop this package into your repo root.

Main entry:
- index.html

Main folders:
- assets/backgrounds/dust9_town
- assets/backgrounds/dust9_town/interiors
- assets/buildings/dust9_town
- assets/props/dust9_town
- assets/ui/dust9_town
- assets/sprites/npcs/dust9_town
- assets/data/dust9_town
- assets/data/dust9_world
- assets/data/dust9_secret_camp
- assets/data/raven_carried_forward

Controls in index:
- T = Dust-9 Town
- L = Saloon
- I = Interior gallery
- 1/2/3 = clear / windy / dust storm
- 4/5/6 = day / dusk / night
- WASD or arrows = camera pan
- + / - = zoom

Notes:
- Town and secret Red Concord camp are separate.
- Raven / earlier Dust-9 camp package is carried forward in assets/data/raven_carried_forward.
- Build JSONs are in assets/data/dust9_town and assets/data/dust9_world.
